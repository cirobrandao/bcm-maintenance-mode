# BCM Maintenance Mode

WordPress plugin to block the front-end for non-admin visitors in dev/test environments.

## Features
- Admin-only access to front-end
- Public visitors see a customizable Maintenance/Development page (no login link)
- Admin bar status indicator (colored dot) with dropdown to switch:
  - Online
  - Maintenance
  - Development

## Install
Copy to:
`wp-content/plugins/bcm-maintenance-mode/`

Activate in WordPress.

## Notes
- Intended for dev/test. Be careful enabling on production without planning.
